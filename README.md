# Machine Learning Projects

A collection of Machine Learning projects built using Python and popular ML libraries to solve real-world prediction, classification, recommendation, and clustering problems.

These projects demonstrate practical applications of data preprocessing, feature engineering, model training, evaluation, and predictive analytics.

---

# Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

# Projects

## 1. Big Mart Sales Prediction
Predicts product sales for Big Mart stores using regression algorithms and historical sales data.

## 2. Breast Cancer Classification
Classifies tumors as malignant or benign using supervised machine learning techniques.

## 3. Calorie Burnt Prediction
Estimates the number of calories burned based on physical and workout-related parameters.

## 4. Customer Segmentation using K-Means Clustering
Groups customers into different clusters based on purchasing behavior and patterns.

## 5. Diabetes Prediction System
Predicts whether a person is diabetic using medical diagnostic parameters.

## 6. Heart Disease Prediction
Analyzes health-related data to predict the likelihood of heart disease.

## 7. House Price Prediction
Predicts house prices based on various property-related features using regression models.

## 8. Loan Status Prediction
Determines whether a loan application is likely to be approved or rejected.

## 9. Movie Recommendation System
Recommends movies to users based on similarity and recommendation algorithms.

## 10. Rock vs Mine Prediction
Classifies sonar signals as either rocks or underwater mines using classification models.

## 11. Spam Mail Prediction System
Detects whether an email message is spam or legitimate using NLP and classification techniques.
